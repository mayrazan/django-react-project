Para rodar o django localmente primeiro instale os pacotes do arquivo requirements.txt com o comando no terminal:
pip install -r requirements.txt

depois de instalado os pacotes rode o servidor com o comando:
python manage.py runserver

para rodar o frontend primeiro entre na pasta frontend pelo terminal através do comando:
cd frontend

dentro da pasta instale os pacotes do arquivo package.json com o comando:
npm install

para isso você precisa ter o node instalado, se não tiver instale e depois rode o comando npm install

depois de instalado os pacotes rode o servidor com o comando:
npm start
